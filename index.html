Projects

Exam Monitoring Assignment Software

2019, Faculdade de Ciências da Universidade de Lisboa

Development of a web application that manages the creation of evaluations and assignment of teachers to oversee the exams of a course. Scored 18,7/20. This project was developed by a team of 5 students on the Software Engineering Project subject. For the development was used the MEAN stack (MongoDB, Express, Node.js and Angular) and the methodology of work was Agile.


Management of Spaces for Sport Events

2019, Faculdade de Ciências da Universidade de Lisboa

Development of an application for managing the use of spaces for sports events. Scored 18/20. This project was developed by a team of 3 students on the course of Construction of Software Systems. The project was done in Java with the platform JavaEE and was composed by a business layer and two clients, a Web Client that used JSP, and a Rich Client Application, developed with JavaFX. Some of the design patterns applied were Data Mapper with JPA, Domain Model, MVC, Front Controller and Stateless Session (using stateless java beans).

Secure Message and File Sharing

2019, Faculdade de Ciências da Universidade de Lisboa

Development of an application for communication and file sharing between clients. Scored 18,6/20. The project was developed in the curricular unit of Security and Dependability in a group of 3 students. The software was developed using sockets for communication, cryptography both symmetric (using RSA) and assymmetric (with TLS/SSL) and the configuration of a firewall.
